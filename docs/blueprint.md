# **App Name**: QuizMaster

## Core Features:

- Quiz Creation: Allow users to create quizzes with multiple question types (multiple choice, true/false, short answer).
- Quiz Taking: Enable users to take created quizzes and receive a score upon completion.
- Question Generation Tool: AI powered feature generates question suggestions and answers for quizzes. LLM acts as a tool to decide whether to include a given piece of information in the response based on a learned decision criteria.
- Quiz Sharing: Allow users to share their created quizzes with others via a unique link.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for sophistication and focus.
- Background color: Very light purple (#F3E5F5).
- Accent color: Blue (#3F51B5) for interactive elements.
- Body font: 'Inter', sans-serif for a modern, neutral look suitable for readability.
- Headline font: 'Space Grotesk', sans-serif to complement the body font with a techy feel
- Use simple, clear icons to represent different quiz categories and actions.
- Clean, organized layout with clear visual hierarchy to guide users through quiz creation and taking processes.
